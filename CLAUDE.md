# 에토스 행정사사무소 — 콘텐츠 파이프라인

## 프로젝트 목표
블로그(네이버) + 링크드인(영문) 운영을 위한 콘텐츠 자동 생성 파이프라인.
글감 입력 → 블로그 초안 + 카드 2장(PNG) + 링크드인 포스트 동시 생성.

## 브랜드 컬러 (항상 적용, 예외 없음)
- 네이비:  #1B2B6B  ← 주 텍스트, 강조, 상단 accent bar
- 골드:    #B8972A  ← 포인트, 장식, badge
- 배경:    #F5F0E8  ← 크림 베이지 (카드 배경)
- 보조:    #FFFFFF  ← 흰 배경 허용
- 텍스트:  #2C2C2A  ← 본문
- 뮤트:    #5F5E5A  ← 캡션, 라벨
- 금지색:  틸 그린, 코럴, 브랜드 외 색상 절대 사용 금지

## 로고
- 파일: assets/logo.png
- 위치: 카드 우하단 워터마크 (투명도 85%)
- 크기: 카드 너비의 18% 이내

## 블로그 글쓰기 원칙
1. 글감 하나당 한 글 — 두 주제 혼합 금지
2. 도입부: 최신 판결/뉴스/통계 훅으로 시작 (웹 검색 필수)
3. 구조: 감정선(공감) → 정보(팩트) → CTA(상담 유도)
4. 수임 연결: "서류로 해결 가능" 메시지 자연 삽입
5. 면책 문구: 본문과 분리, 마지막 배치
6. 출처: 모든 팩트 웹 검색 검증 후 URL 포함

## 카드 원칙
- 인스타그램: 1080×1080px
- 링크드인:   1200×628px
- 카드당 핵심 메시지 1개, 텍스트 최소화
- 정보는 표/아이콘으로, 설명은 키워드로
- 로고 항상 포함, 출처 표기 필수
- 제네릭 레이아웃 금지

## 링크드인 원칙
- 영문 작성
- 후킹 첫 줄 (스크롤 멈추게)
- 태그: 영문 위주 15개 이내
- CTA: 단일 행동 유도

## 워크플로우 (입력 형식)
```
글감: [주제]
독자: [타깃]
훅:   [최신 판결/뉴스/통계]
상황: [독자 처한 상황]
출력: [blog / card / linkedin / all]
```

## 스킬 참조
- skills/research.md  ← 팩트 검증 원칙
- skills/marketing.md ← SEO·CTA 원칙
- skills/taste.md     ← 카드 디자인 원칙
- DESIGN.md           ← 브랜드 디자인 시스템

## 출력 경로
- outputs/blog/     ← 블로그 .md 파일
- outputs/cards/    ← 카드 PNG 파일
- outputs/linkedin/ ← 링크드인 .md 파일
