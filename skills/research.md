# Research & Fact Verification Skill

## 핵심 원칙
모든 팩트는 게시 전 웹 검색으로 검증한다.
확인 불가한 수치·판결·통계는 작성하지 않는다.

## 검증 단계
1. **훅 검증**: 도입부 판결/뉴스는 반드시 원문 URL 확인
2. **수치 검증**: 통계·금액·날짜는 공식 출처(법원, 정부, 공신력 있는 기관) 확인
3. **판례 검증**: 사건번호, 법원명, 선고일 3가지 모두 확인
4. **출처 표기**: 본문 내 출처 명시 또는 각주 처리

## 출처 우선순위
1. 법원 판결문 / 대법원 홈페이지
2. 정부 공식 사이트 (법제처, 식품의약품안전처 등)
3. 주요 언론 (연합뉴스, 헤럴드경제, 법률신문)
4. 학술·연구기관 보고서

## 금지 행위
- 기억에만 의존한 수치 사용
- "~라고 알려져 있다" 식 모호한 표현으로 검증 우회
- 오래된 판례를 최신인 것처럼 제시
