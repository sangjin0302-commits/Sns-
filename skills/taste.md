# Design Taste Skill (taste-skill 기반)

## 핵심 철학
여백이 정보만큼 중요하다.
색은 의미를 인코딩하지 장식하지 않는다.
한 장에 한 메시지.

## 레이아웃 원칙
- 시각적 계층: 헤드라인 1개 > 서브 1개 > 바디
- 여백: 콘텐츠 영역의 최소 20%는 여백
- 정렬: 왼쪽 정렬 기본 (가독성 최우선)
- 그리드: 72px 패딩, 일관된 간격

## 색 사용 원칙
- 카드당 최대 2가지 색 (네이비 + 골드)
- 배경: 크림(#F5F0E8) 또는 흰색
- 강조: 골드는 1-2곳만
- 아이콘: 네이비 또는 골드만

## 타이포그래피
- 헤드라인: 굵게, 네이비, 40-50px
- 서브: 보통, 네이비, 24-28px
- 바디: 보통, 다크그레이, 20-22px
- 캡션: 작게, 뮤트, 18-20px

## Anti-slop 규칙 (taste-skill)
- 제네릭한 "깔끔한" 레이아웃 금지
- 스톡 이미지 느낌 금지
- 모든 요소는 브랜드 컬러와 연결
- 로고 없는 카드 금지
- 출처 없는 카드 금지

## 카드 품질 체크리스트
- [ ] 브랜드 컬러만 사용했나?
- [ ] 메시지가 1개로 명확한가?
- [ ] 로고 포함됐나?
- [ ] 출처 표기됐나?
- [ ] 여백이 충분한가?
- [ ] 텍스트 계층이 명확한가?
