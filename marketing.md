# Marketing & Content Skill

## SEO 원칙 (marketingskills 기반)
- 제목: 독자가 실제 검색할 키워드 포함
- 소제목: 롱테일 키워드 자연 삽입
- 메타 설명: 120자 이내, 검색 의도 명확히
- 키워드 밀도: 자연스럽게, 과도한 반복 금지

## 콘텐츠 구조 (AARRR 중 Activation 집중)
```
Awareness   → 훅 (판결/뉴스로 관심 유발)
Interest    → 공감 (독자 상황 묘사)
Desire      → 정보 (해결책 제시)
Action      → CTA (상담 신청 유도)
```

## CTA 원칙
- 글당 CTA 1개만
- 구체적 행동: "상담 신청" > "문의하기" > "알아보기"
- 위치: 본문 마지막, 면책 문구 직전
- 톤: 강요 아닌 자연스러운 연결

## 태그 전략
- 네이버 블로그: 한글 태그 10-15개, 검색량 높은 단어 우선
- 링크드인: 영문 태그 15개 이내, 전문 분야 + 이슈 혼합
- 혼합 예시: #행정사 #동거계약서 #비혼커플 + #KoreanLaw #AdminLaw

## 발행 빈도 권장
- 블로그: 주 1-2회 (품질 > 양)
- 링크드인: 블로그 발행 시 동시 발행
- 카드뉴스: 블로그 1개당 카드 2장
